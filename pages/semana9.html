¿Qué es un Router?

React Router

Es un conjunto de componentes de navegación para:
- Establecer rutas en nuestra aplicación web
- Realizar redirecciones a otras páginas según requerimientos
- Crear páginas públicas y privadas
- Acceder al historial de navegación
- Gestionar rutas 404

Instalando React Router en un Proyecto

Para instalar usamos el comando NPM

>npm install -D react-router-dom

Luego estructuramos el proyecto.
