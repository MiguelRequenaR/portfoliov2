Manejo del BOM y DOM

BOM ¿Qué es BOM?

El Browser Object Model (BOM) es una API (Application Programming Interface) proporcionada por los navegadores web para interactuar y controlar elementos del navegador y la ventana del navegador.

El BOM proporciona objetos y métodos para trabajar con elementos como la ventana del navegador, los marcos (frames), los historiales de navegación, las cookies, las ubicaciones y los eventos del navegador.

DOM ¿Qué es DOM?

El DOM, o Document Object Model (Modelo de Objetos del Documento), es una representación estructurada en forma de árbol de un documento HTML o XML.

El DOM representa cada elemento del documento (como etiquetas HTML) como un objeto en el árbol, permitiendo a los desarrolladores web interactuar con ellos de manera programática. Esto significa que se pueden agregar, eliminar o modificar elementos y contenido en la página web utilizando JavaScript u otros lenguajes de programación compatibles con el DOM.

¿Para que sirve DOM?

El DOM (Document Object Model) sirve para permitir la interacción y manipulación de elementos y contenido en una página web:

1. Manipulación del contenido: El DOM permite acceder y modificar el contenido de una página web, como texto, imágenes, enlaces, formularios, etc.

2. Manipulación de estilos: Con el DOM, se puede acceder y modificar los estilos CSS aplicados a los elementos de la página. Esto permite cambiar el diseño, los colores, las fuentes y otros aspectos.

3. Manipulación de la estructura: El DOM permite agregar, eliminar y reorganizar elementos en la página. Esto es útil para crear o eliminar contenido de forma dinámica, como agregar nuevos elementos HTML.

4. Manipulación de eventos: El DOM facilita la interacción con eventos del usuario, como hacer clic en un botón, mover el mouse, escribir en un formulario, etc. Permite definir acciones que se ejecuten en respuesta a estos eventos y manipular la página en consecuencia.

5. Acceso a APIs externas: El DOM proporciona acceso a otras APIs web, como la API de geolocalización, la API de cámara, la API de notificaciones, entre otras. Esto permite integrar funcionalidades externas en la página web y brinda capacidades más avanzadas a las aplicaciones web.

Trabajo de laboratorio

Desarrollar la animación de una pelota, que tenga 4 botones, cada boton es una acción diferente, los botones serán respectivamente: Play, Pause, STOP, y Reverse.

Comentario sobre la clase:

El DOM es una parte esencial en el desarrollo web, ya que brinda a los desarrolladores la capacidad de manipular y controlar de forma dinámica el contenido y la apariencia de una página web. Esto permite crear experiencias interactivas y personalizadas para los usuarios.
